# CROW1

Welcome to the Tkinter version of CROW. Here you will find everything you need
to get started.

The instructions folder contains guidance on the setup and use of CROW1.

If you have additional questions, or issues, please raise an issue in the
repository or contact: <linkage.hub@ons.gov.uk>

## Data

The data included in the `data` folder is fake.

The `pairwise_example_data.csv` dataset is laid out in the format needed by the
pairwise version of CROW1, whereas the `clusters_example_data.csv` dataset is
laid out in the format used by the cluster version of CROW1.
